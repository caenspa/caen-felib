92v1.2.0 (28/03/2022)
93-------------------
94
95New features:
96- New CAEN_FELib_HasData that checks if an endpoint node has data to be read
97 with a subsequent call to CAEN_FELib_ReadData.
98
99Changes:
100- CAEN_FELib_nodeType_t removed (deprecated since v1.1.4).
101
102Fixes:
103- Resource version properly set for Windows DLL, setting both file version and
104 product version fields to the library version.
105- Removed static library generation from autoconf since currently the library
106 requires the constructor and descructors to be invoked at load.
107- Removed useless includes.
108- Fix CAEN_FELib_Open failure on macOS due to different shared library file
109 extensions.
110- Removed workaround for libstdc++ bug 67791 when compiling for macOS because
111 it was not working. Actually, it is not required on that target OS.
