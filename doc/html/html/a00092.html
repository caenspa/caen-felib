◆ CAEN_FELib_Open()

int CAEN_FELib_Open ( const char *  url,
uint64_t *  handle 
)

Connect to a device.

Parameters
[in]urlURL of device to connect (null-terminated string, with standard format [scheme]:[//authority][path][?query][#fragment])
[out]handleroot handle, representing a node of type CAEN_FELib_DIGITIZER
Return values
CAEN_FELib_Success(0) in case of success
CAEN_FELib_BadLibraryVersionin case of success, but using a old version of the underlying library that may not support some new features of the hardware
ora negative error code specified in CAEN_FELib_ErrorCode
Note
When returning CAEN_FELib_BadLibraryVersion the connection has been properly established, but there could be unexpected behaviours: the user should update the underlying library as soon as possible.
Warning
CAEN_FELib_Open() and CAEN_FELib_Close() modify a static variable: are not thread safe.
